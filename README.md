# Pilha Mínima e Máxima

=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=
Código de demonstração de um sistema de stack que, além de permitir adicionar e remover elementos, retorna os maiores e menores elementos da pilha, sem removê-los.
Inclui uma demonstração automática, bem como um menu interativo que permite ao usuário inserir seus próprios valores e executar as funções que bem deseja.

NOTA: para rodar o código com uma demonstração automática;
javac MinMaxStack.java MinMaxStackDemo.java
java MinMaxStackDemo.java

Para rodar o código podendo inserir seus próprios valores nele;
javac MinMaxStack.java MinMaxStackDemo.java
java MinMaxStack.java
